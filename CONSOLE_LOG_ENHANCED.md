# Console Log แสดงรายละเอียดการคำนวณรายรับขายฝาก

## การปรับปรุง

เพิ่ม console.log ที่แสดงรายละเอียดการคำนวณรายรับขายฝากอย่างชัดเจน พร้อม emoji และรูปแบบที่อ่านง่าย

## ตัวอย่าง Console Log ที่จะเห็น

```
═══════════════════════════════════════════════════════════
📊 สรุปรายรับขายฝาก (Pawn Income Summary)
═══════════════════════════════════════════════════════════
1️⃣  ดอกเบี้ย (Interest):
   - จำนวนเงิน: ฿3,600
   - มาจาก: รายการหักดอก/ต่อดอกในเดือนนี้

2️⃣  ยอดไถ่ถอน (Redemption Amount):
   - จำนวนเงิน: ฿1,200
   - มาจาก: ลูกค้ารับเครื่องคืนในเดือนนี้

💰 รายรับขายฝากรวม = ดอกเบี้ย + ยอดไถ่ถอน
    ฿3,600 + ฿1,200 = ฿4,800

✅ รายรับขายฝากที่แสดงในการ์ด: ฿4,800
═══════════════════════════════════════════════════════════
```

## สิ่งที่แสดง

### 1. ดอกเบี้ย (Interest)
- แสดงจำนวนเงินดอกเบี้ยที่ได้รับ
- อธิบายว่ามาจากรายการหักดอกหรือต่อดอกในเดือนนั้น

### 2. ยอดไถ่ถอน (Redemption Amount)
- แสดงจำนวนเงินที่ได้จากการรับเครื่องคืน
- อธิบายว่ามาจากลูกค้ารับเครื่องคืน

### 3. การคำนวณ
- แสดงสูตร: รายรับขายฝาก = ดอกเบี้ย + ยอดไถ่ถอน
- แสดงตัวเลขจริง: ฿3,600 + ฿1,200 = ฿4,800

### 4. ผลลัพธ์
- แสดงยอดสุดท้ายที่จะไปแสดงในการ์ดรายรับขายฝาก

## วิธีดู Console Log

1. เปิดหน้าแดชบอร์ด
2. กด `F12` เพื่อเปิด Developer Tools
3. ไปที่แท็บ **Console**
4. จะเห็น log นี้แสดงอยู่ตอนโหลดแดชบอร์ด

## ประโยชน์

- ✅ เห็นการคำนวณอย่างชัดเจน
- ✅ ตรวจสอบว่ายอดไถ่ถอนถูกนำมารวมหรือไม่
- ✅ Debug ได้ง่ายถ้ามีปัญหา
- ✅ อ่านง่าย มี emoji ช่วยจำแนกส่วนต่างๆ

## ตัวอย่างกรณีต่างๆ

### กรณี 1: มีทั้งดอกเบี้ยและยอดไถ่ถอน
```
1️⃣  ดอกเบี้ย: ฿3,600
2️⃣  ยอดไถ่ถอน: ฿1,200
💰 รวม: ฿4,800
```

### กรณี 2: มีแต่ดอกเบี้ย (ไม่มีคนรับเครื่องคืน)
```
1️⃣  ดอกเบี้ย: ฿3,600
2️⃣  ยอดไถ่ถอน: ฿0
💰 รวม: ฿3,600
```

### กรณี 3: มีแต่ยอดไถ่ถอน (ไม่มีดอกเบี้ย)
```
1️⃣  ดอกเบี้ย: ฿0
2️⃣  ยอดไถ่ถอน: ฿1,200
💰 รวม: ฿1,200
```

## ไฟล์ที่อัพเดท

- ✅ `script.js` - เพิ่ม console.log ที่ละเอียดและอ่านง่าย

## หมายเหตุ

- Console log นี้จะแสดงทุกครั้งที่โหลดหรือ refresh แดชบอร์ด
- ใช้ emoji เพื่อให้อ่านง่ายและแยกส่วนได้ชัดเจน
- แสดงทั้งตัวเลขและคำอธิบาย
